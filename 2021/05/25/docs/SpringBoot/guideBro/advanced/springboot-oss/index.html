1.1 前置知识

具有 Java 基础以及SpringBoot 简单基础知识即可。

1.2 环境参数

  • 开发工具：IDEA
  • 基础工具：Maven+JDK8
  • 所用技术：SpringBoot+阿里云OSS 存储服务 Java 相关API
  • SpringBoot版本：2.1.0

  • 如何自己搭建一个图床使用（通过前端选择图片，支持预览，但不支持修改图片）
